Guida all'installazione
Come configurare Magic Vault sul tuo telefono e PC
1 Struttura dell'app
Magic Vault ha ora 5 sezioni separate:
  1. Ricerca — cerca qualsiasi carta su Scryfall e scegli dove aggiungerla
  2. Collezione — il tuo inventario personale di carte fisiche
  3. Set — traccia le carte di ogni espansione separatamente (es. "ho questa carta nel booster del set X ma non in collezione")
  4. Mazzi — lista dei mazzi costruiti con verifica delle carte mancanti
  5. Guida — questa pagina
Collezione vs Set: Puoi tenere separate le carte "sfuse" in collezione dalle carte che hai aperto nei booster di un set specifico. Una carta può essere presente in entrambi, solo in uno, o in nessuno.
2 Come aggiungere carte
Dalla sezione Ricerca, cerca la carta e per ogni risultato hai tre pulsanti:
  1. + Collezione — aggiunge la carta alla tua collezione generale
  2. + Set — aggiunge la carta al set/espansione di appartenenza (inventario separato)
  3. Dettagli — apre il pannello completo con anteprima immagine, scelta edizione, condizione ecc.
3 Google Drive — struttura cartelle
I dati vengono salvati in MagicVault/ su Google Drive:
  1. collection.json — collezione generale
  2. sets.json — set tracciati + inventario carte per set
  3. decks.json — mazzi
Automatico: la cartella viene creata al primo salvataggio.
4 Connetti Google Drive
  1. Apri il file in Chrome o Edge
  2. Clicca "Connetti Drive" in alto a destra
  3. Accedi con Google — pallino verde ✅ = connesso
5 Installazione Android
  1. Apri il file in Chrome su Android
  2. Menu tre puntini → Aggiungi alla schermata Home
  3. Appare come app!
6 Installazione PC
  1. Apri in Chrome/Edge → icona ⊕ Installa nella barra indirizzi
  2. Oppure Edge: menu → App → Installa sito come app
7 Client ID Google (richiesto per Drive)
  1. Vai su console.cloud.google.com
  2. Nuovo progetto → Magic Vault
  3. API e Servizi → Libreria → abilita Google Drive API
  4. Credenziali → Crea → ID client OAuth 2.0 → Applicazione web
  5. In "URI JavaScript autorizzati" aggiungi:
    https://tuonome.github.io
    (sostituisci "tuonome" con il tuo username GitHub)
  6. Copia il Client ID, incollalo qui sotto → clicca Salva
  7. Clicca "Connetti Drive" in alto → si apre il popup Google → accedi
⚠ Non aprire il file con doppio click! Google blocca l'autenticazione da file locali. Usa sempre l'URL di GitHub Pages nel browser.
